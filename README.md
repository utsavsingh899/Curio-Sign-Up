# Curio-Sign-Up
A simple website for signing up to my mailing list where I'll be notifying all about my new articles on [Curio](https://medium.com/curio). Visit the website [here](https://curio-sign-up.herokuapp.com/).
### Technologies:
1. HTML/CSS
2. Bootstrap
3. Node + Express
4. Mailchimp API
5. Hosted on Heroku
